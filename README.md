# ads2arab

A WebView mobile application generated by Golden WebView App WordPress Plugin.

## Configuration

- **Site URL**: https://ads2arab.com/
- **App ID**: com23.ads2arab.co

## Building

### Using Codemagic

1. Push this repository to GitHub
2. Connect to Codemagic
3. Select the workflow (android-build or ios-build)
4. Start the build

### Local Development

```bash
flutter pub get
flutter run
```

## Environment Variables

The app reads configuration from `assets/config/app_config.json` or from dart-define arguments during build.
